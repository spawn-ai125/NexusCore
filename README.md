## 🛠️ Nasıl Kullanılır?

1. **NexusCoreOrigin.cs** dosyasını `Assets/Editor` klasörüne kopyalayın.
2. Unity üst menüsünden `NexusCore > Origin Engine` yolunu izleyin.
3. Hayalinizdeki oyunu yazın ve **GENERATE UNIVERSE** butonuna basın.
